```

  - Blue (#3264C8):        Primary actions, links
  - Green (#32C864):       Success, resume, active
  - Amber (#FFC832):       Warning, paused, attention
  - Red (#C83232):         Error, danger, stop

Background Colors:
  - Dark Background (#1E1E28):     Main background
  - Medium Dark (#28283C):         Panel backgrounds
  - Dark Accent (#3C3C46):         Disabled states, borders

Text Colors:
  - Light Text (#DCDCDC):          Primary text
  - Medium Text (#A0A0A0):         Secondary text
  - Dim Text (#787878):            Disabled text

  - Purple (#C864FF):              Special highlights
  - Cyan (#64C8FF):                Links, info
  - Orange (#FF9632):              Warnings
```

## Responsive Breakpoints

```
Large Desktop (1920px+):
  - Full 4-panel layout
  - All panels visible
  - Maximum information density

Desktop (1280px - 1919px):
  - 3-panel layout
  - Thought and Context side-by-side
  - Breakpoints below
  - Time travel collapsible

Laptop (1024px - 1279px):
  - 2-panel layout
  - Stacked panels
  - Collapsible sections
  - Compact controls

Tablet (768px - 1023px):
  - Single column layout
  - Tabbed interface
  - Essential controls only
  - Full-width panels
```

## Accessibility Features

```
Keyboard Navigation:
  - Tab: Navigate between controls
  - Enter: Activate button/action
  - Space: Toggle checkbox
  - Arrow keys: Navigate timeline
  - F-keys: Debugger shortcuts

Screen Reader Support:
  - ARIA labels on all interactive elements
  - Live regions for status updates
  - Semantic HTML structure
  - Role attributes

Visual Accessibility:
  - High contrast mode support
  - Configurable font sizes
  - Color-blind friendly palette
  - Focus indicators
  - Sufficient color contrast ratios (WCAG AAA)
```

```

  - Panel resize: 200ms ease-in-out
  - Tab switch: 150ms ease-in
  - Button hover: 100ms ease-out
  - Status change: 300ms ease-in-out

Attention Animations:
  - Breakpoint hit: Pulse effect (500ms)
  - New event: Fade in (200ms)
  - Error state: Shake effect (300ms)
  - Step execution: Progress indicator
```
